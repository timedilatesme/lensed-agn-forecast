# Lensed AGN Forecasts

This repository uses `slsim` to simulate and forecast the yields and physical properties of strongly lensed AGNs (quasars) across upcoming astronomical surveys, including Rubin LSST and the Roman Space Telescope (Medium, Wide, and NANCY tiers).
